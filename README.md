# PrayerZones

**Official Repository for the PrayerZones Android Application.**

PrayerZones is a dedicated utility application designed for the religious community to track prayer timings and manage zone-based notifications. 

### Project Purpose
This application is provided as a free community tool. We utilize Stripe integration to allow users to provide **voluntary tips** and contributions. These funds are used exclusively to:
* Maintain cloud server hosting.
* Cover API subscription costs for precise location data.
* Support ongoing independent development.

### Business Model
* **No Paywalls:** All core features are free to the public.
* **Support-Based:** Payments are non-mandatory contributions to keep the service running.
* **Developer:** Managed by **Abang-App**.
